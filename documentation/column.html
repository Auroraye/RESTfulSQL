Column

This model is designed for modify columns in a table after the creation of the table.

The GET method is for querying the information of the columns of a specified table. The UPDATE method is for updating the properties of the specified columns, and the operation including: rename column, set default value, set value type, set nullability, add unique keys, and remove unique keys. The POST and DELETE method are simply insert and drop columns from a table, and they are similar with this method.
